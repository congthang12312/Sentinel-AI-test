Write high-converting copy using proven formulas, templates, and writing styles.

## Copy Formulas

| Formula | Structure | Best For |
|---|---|---|
| **AIDA** | Attention → Interest → Desire → Action | Landing pages, ads |
| **PAS** | Problem → Agitate → Solution | Email, sales pages |
| **BAB** | Before → After → Bridge | Testimonials, case studies |
| **4Ps** | Promise → Picture → Proof → Push | Long-form sales |
| **4Us** | Urgent + Unique + Useful + Ultra-specific | Headlines |
| **FAB** | Feature → Advantage → Benefit | Product descriptions |

## Workflow

### Step 1: Understand the Brief
- What type of content? (headline, email, landing page, social, product description)
- Target audience? Awareness level?
- Tone? (professional, casual, urgent, friendly)
- Key benefit / unique selling point?
- Call to action?

### Step 2: Research
- Check `references/writing-styles.md` in copywriting skill for 30+ styles
- Read `references/copy-formulas.md` for formula details
- Check brand guidelines if they exist in `./docs/`

### Step 3: Write
- Apply the most appropriate formula based on content type
- Lead with benefit, not feature
- Be specific — avoid vague claims
- One CTA per piece
- Match copy to audience awareness level

### Step 4: Refine
- Read aloud — if awkward, rewrite
- Test multiple headline variations
- Ensure consistency with brand voice
- Check for: clarity, urgency, specificity, credibility

### Step 5: Deliver
- Present 2-3 variations for key elements (headlines, CTAs)
- Explain the formula used and why
- Suggest A/B testing strategy
