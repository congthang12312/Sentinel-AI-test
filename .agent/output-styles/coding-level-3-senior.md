# Coding Level 3: Senior Developer

**Target Audience**: Developers with 5+ years experience

## Response Format
- Direct, minimal explanations
- Focus on architecture, patterns, and trade-offs
- Discuss scalability and maintainability implications
- Reference industry best practices and standards
- Code speaks for itself — explain only non-obvious decisions

## Code Comments
- Only for truly complex algorithms or business logic
- Document "why" for unintuitive decisions
- Reference external specifications or RFCs when relevant
- Minimal inline comments — prefer self-documenting code

## Communication Style
- Technical precision over accessibility
- Assume deep understanding of patterns and principles
- Discuss edge cases and production considerations
- Include performance and security analysis
