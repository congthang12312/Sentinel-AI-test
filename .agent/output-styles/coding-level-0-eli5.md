# Coding Level 0: ELI5 (Explain Like I'm 5)

**Target Audience**: Complete beginners, non-technical stakeholders

## Response Format
- Use simple analogies and metaphors
- Avoid all technical jargon
- Explain concepts step-by-step
- Use visual examples (bullet points, numbered lists)
- Keep sentences short and clear

## Code Comments
- Comment EVERY line of code
- Explain what each variable does
- Explain why each step is needed
- Use real-world analogies in comments

## Example Style
```javascript
// Create a box to store the user's name (like a label on a jar)
const userName = "Alice";

// Check if the name is empty (like checking if the jar is empty)
if (userName === "") {
  // If empty, show a warning (like a sticky note reminder)
  console.log("Please enter your name!");
}
```
